Project Overview
A modern, user-friendly Flashcard Quiz App built with Kotlin and Jetpack Compose. This application is designed to help users study efficiently by creating custom digital flashcards with a clean and intuitive interface.

Features
Create Flashcards: Users can easily add new cards with specific questions and answers.

Study Mode: Simple navigation between cards using "Next" and "Previous" buttons.

Toggle Answers: A dedicated "Show/Hide Answer" button to test knowledge before revealing the result.

Manage Content: Full support for editing and deleting existing cards for personalized learning.

Modern UI: Built using Material 3 components for a sleek and responsive user experience.

Technical Stack
Language: Kotlin.

UI Framework: Jetpack Compose (Declarative UI).

Architecture: MVVM (Model-View-ViewModel) for clean code separation.

State Management: State hoisting and ViewModel for persistent UI state.

How to Run
Clone this repository.

Open the project in Android Studio (Ladybug or newer).

Build and run on an emulator or physical device.
